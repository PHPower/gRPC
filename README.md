# `gRPC`官方文档中文版，个人翻译学习使用，以`Golang`为基础

`gRPC`是一个现代的，开源的远程过程调用（`RPC`）框架，可以在任何地方运行。它使客户端和服务器应用程序可以透明地进行通信，并使构建连接的系统更加容易。
